Install notes

macOS: open the DMG, drag Sitrec to Applications, then launch from Applications.

Windows: run the Setup installer and follow the prompts.

macOS

Download the DMG, open it, drag Sitrec into Applications, then launch it from Applications.

If macOS warns before first launch, Control-click the app in Applications and choose Open.

Windows

Download the Setup installer, run it, choose an install location if needed, and finish the installer.

After installation, launch Sitrec from the Start menu or the desktop shortcut.
